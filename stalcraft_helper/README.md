# STALCRAFT X — помощник перекупщика

Маленькое standalone-приложение на PyQt6 для тех, кто крутит лоты и
варит катализаторы в STALCRAFT X. Окно работает отдельно от игры,
ничего никуда не инжектится, кода игры не касается — это просто
калькулятор + список карточек.

## Что внутри

### Вкладка «Основное перекупство»

Сетка карточек предметов. На каждой:

- Картинка (выбирается из файла, копируется в `~/.stalcraft_helper/images/`).
- Название.
- Цена за штуку и количество.
- Сумма по карточке (цена × количество).

Внизу — общая сумма всех карточек и кнопка **«Копировать»**, чтобы
быстро вставить итог в чат / в заявку.

### Вкладка «Катализаторы»

Калькулятор себестоимости катализатора по рецептам:

| Рецепт | Ингредиенты | Выход |
| --- | --- | --- |
| Светящийся сахар | 100 аномальной пыли · 10 мякоти сластены · 1 аномальная плазма | 30 шт |
| Катализатор     | 15 светящегося сахара · 100 аномальной пыли                    | 20 шт |

Ты вводишь цены за единицу аномальной пыли / мякоти сластены /
аномальной плазмы — приложение считает себестоимость 1 светящегося
сахара, 1 катализатора и партии в 100 катализаторов, плюс показывает,
сколько каждого сырья нужно. Ниже отдельное поле для цены продажи
100 катализаторов — снизу выводится прибыль и ROI.

## Запуск

```bash
pip install -r stalcraft_helper/requirements.txt
python -m stalcraft_helper
```

Состояние (цены, карточки, выбранные картинки) сохраняется в
`~/.stalcraft_helper/state.json` и подгружается при следующем запуске.

## Дисклеймер

Это внешний инструмент. Он не читает память игры, не подключается к
процессу `stalcraft.exe` и не использует никаких хуков. Можно держать
поверх игры через стандартное `Always on top` оконного менеджера или
вторым монитором.
